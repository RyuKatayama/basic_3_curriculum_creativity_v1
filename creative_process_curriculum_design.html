創造のプロセス

12週間学習カリキュラム - 全体設計書
夢・アーキテクティング工房（Dream Architecting Manufactory）
📚 カリキュラム概要
このカリキュラムは、「創造」という人間の最も高度な知的活動を、
理論と実践の両面から体系的に学ぶ12週間プログラムです。

単なる「創造性テクニック」の習得にとどまらず、
創造プロセスの本質的な理解、情報幾何学的な数理的基盤、
そして実際の創造活動への応用までを網羅します。

対象者：
• イノベーターを目指す実務家
• 研究者・エンジニア
• クリエイター・アーティスト
• 起業家・事業開発者
• 教育者
• 創造的活動に関心のあるすべての方

前提知識

必須：
• 特になし（好奇心と学ぶ意欲があれば十分）

あると望ましい：
• 何らかの創造的活動の経験（趣味レベルでも可）
• 問題解決の経験
• 基礎的な論理的思考力

Phase 2以降で役立つ：
• 高校数学レベルの知識（微分・確率の基礎）
• プログラミング経験（実習を深めたい場合）

💡 重要： 数学的な内容は含まれますが、直感的理解を重視します。
数式が苦手な方でも、本質的な理解は十分可能な構成になっています。

到達目標

このカリキュラムを修了すると、以下ができるようになります：

1. 理論的理解：
• 創造プロセスの発散・収束モデルを説明できる
• 情報幾何学的な視点で創造を理解できる
• 創造における停止条件を判断できる
• パレート最適化の概念を創造に応用できる

2. 実践的スキル：
• 効果的な発散思考を実践できる
• 適切な収束判断ができる
• イノベーティブ思考法を使いこなせる
• 創造的プロジェクトを設計・実行できる

3. メタ認知能力：
• 自分の創造プロセスを観察・改善できる
• 創造活動における自分の強み・弱みを理解できる
• 状況に応じて思考法を切り替えられる

4. 統合的視点：
• 創造・イノベーション・問題解決を統一的に理解できる
• 個人の創造性と組織のイノベーションを関連づけられる
• 創造性を人生の様々な場面に応用できる

カリキュラム構造

Phase 0（第1-3週）：創造の基礎
創造とは何か、創造プロセスの全体像、基本的な思考法
Phase 1（第4-6週）：発散思考の深化
アイデア生成、発散的思考法、可能性の探索
Phase 2（第7-9週）：収束思考と最適化
評価・選択、収束的思考法、意思決定
Phase 3（第10-12週）：統合と実践
発散・収束の統合、実践プロジェクト、継続的創造

• 創造プロセスにおける「揺らぎ」の重要性

• 創造における「インキュベーション」

図解：
• 創造プロセスの可視化
• 発散・収束サイクル
• ダブルダイヤモンドモデル

実習：
• 簡単な創造課題の実践
• 自分のプロセスの観察
• プロセス日記の開始
📅 第3週：創造的思考の基礎
学習内容：
• 論理的思考 vs 創造的思考
• 垂直思考と水平思考（de Bono）
• 批判的思考と創造的思考の関係
• メタ認知と創造性
• 思考の柔軟性を高める方法
• 固定観念からの脱却

演習：
• 9点問題
• 固定観念チェック
• リフレーミング練習
• アナロジー思考

実習：
• 日常の中で前提を疑う練習
• 「なぜ？」を5回繰り返す
• 異なる視点からの観察
🎯 Phase 0 到達目標

• 発散の深さと広さのバランス
• 早すぎる収束の危険性

• アイデアの分布

💡 重要な方針：
このカリキュラムは「正解」を教えるものではありません。
創造に「唯一の正解」はなく、あなた自身の創造スタイルを見つけることが目標です。
評価も、「正解への到達度」ではなく、「学びの深さ」と「実践の真摯さ」を重視します。
